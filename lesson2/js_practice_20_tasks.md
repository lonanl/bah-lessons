## ℹ️Справка: Шаблонные строки

В JavaScript есть удобный способ собирать строки из текста и переменных — шаблонные строки. Они пишутся не в обычных кавычках ' или ", а в обратных кавычках:
```js
let name = "Алексей";
console.log(`Привет, ${name}!`); \\ Выведет "Привет, Алексей"
```
Главная особенность: внутрь строки можно вставлять переменные и выражения через ${...}.
Например:
```js
let age = 15;
let birthYear = 2026 - age;

console.log(`Тебе ${age} лет. Ты родился примерно в ${birthYear} году.`);
```
вместо:
```js
console.log("Тебе " + age + " лет. Ты родился примерно в " + birthYear + " году.");
```
Они помогают быстро и удобно собирать длинные строки, в которые надо динамически встроить переменные или выражения. Мы их указываем через `${выражение/переменная}` - доллар и фигурные скобки - и во время выполнения того где мы используем эту строку, на место это конструкции подставляется выражение/переменная.

**!Важно такая строка пишется в обратных кавычках ``` `строка` ```, не в обычных или двойных!!** Ставится с помощью кнопки где русская буква ё (только на английской раскладке), под кнопкой escape

Это особенно удобно, когда нам надо вставить в строку сразу несколько переменных. В идеале - использовать их вместо `строка + строка`

## ℹ️Справка: метод `.trim()`

Часто пользователь случайно вводит строку с лишними пробелами в начале или в конце:

```txt
"   Алексей   "
```

Для программы это не то же самое, что:

```txt
"Алексей"
```

Чтобы убрать пробелы в начале и конце строки, можно использовать метод `.trim()`. Мы вызываем его через точку после строки (или содержащих ее переменной или выражения)

```js
let name = "   Алексей   ";
let cleanName = name.trim();

console.log(cleanName); // Выведет "Алексей"
```

Важно: `.trim()` не изменяет старую строку, а возвращает новую очищенную строку.

```js
let text = "   привет   ";
text = text.trim();
```

Подробнее:

- Строки в JavaScript: https://learn.javascript.ru/string
- `.trim()` на MDN: https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String/trim

## ! Справка: `.toLowerCase()` и `.toUpperCase()`

Пользователь может вводить текст в разном регистре:

```txt
"да"
"Да"
"ДА"
"дА"
```

Для человека это одно и то же, а для JavaScript — разные строки.

Чтобы привести строку к нижнему регистру, используется `.toLowerCase()`:

```js
let answer = "ДА";
answer = answer.toLowerCase();

console.log(answer); // "да"
```

Чтобы привести строку к верхнему регистру, используется `.toUpperCase()`:

```js
let city = "moscow";
city = city.toUpperCase();

console.log(city); // "MOSCOW"
```

Часто методы можно использовать цепочкой:

```js
let answer = "   ДА   ";
answer = answer.trim().toLowerCase();

console.log(answer); // "да"
```

Подробнее:

- Строки в JavaScript: https://learn.javascript.ru/string
- `.toLowerCase()` на MDN: https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String/toLowerCase
- `.toUpperCase()` на MDN: https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String/toUpperCase

## ! Справка: метод `.includes()`

## ℹ️ Справка: метод `.slice()`

Метод `.slice()` позволяет взять часть строки.

```js
let word = "JavaScript";

console.log(word.slice(0, 4)); // "Java"
console.log(word.slice(4));    // "Script"
```

Первое число — индекс символа, откуда начать считывать (начиная с нуля то есть с n-1 символа, потому что в программировании все считают снуля). Второе число — конечный символ невключительно.

Можно указать только откуда начать, тогда строка выберется до конца.

```js
let name = "Алексей";

console.log(name.slice(0, 1)); // "А" - с самого начало по 1-й символ не включая (1-й то есть реально второй, т.к. реально первый это 0)
console.log(name.slice(1));    // "лексей"
```

Еше раз про индексы: в программировании индекс - порядоквый номер элемента в массиве или в строке. Индексы начинаются с 0. То есть для строки `"Алексей"` индексы будут такими:

| Индекс | Значение |
|--------|----------|
| 0      | "А"      |
| 1      | "л"      |
| 2      | "е"      |
| 3      | "к"      |
| 4      | "с"      |
| 5      | "е"      |
| 6      | "й"      |

Соответственно если мы хотим получить подстроку `лекс`, нам нужно с индекса 1 по индекс 4, а так как в .slice возвращает подстроку не включая символ второго индекса, то 4+1=5 итого `"Алексей".slice(1, 5) // лекс`

Подробнее:

- Строки в JavaScript: https://learn.javascript.ru/string
- `.slice()` на MDN: https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String/slice

## ℹ️ Справка: цикл `while`

Иногда нужно повторять действие несколько раз.

Например, спрашивать пароль до тех пор, пока пользователь не введёт правильный:

```js
let password = prompt("Введите пароль");

while (password !== "12345") {
  password = prompt("Неверно. Попробуйте ещё раз");
}

console.log("Доступ открыт");
```

Цикл `while` работает так:

```js
while (условие) {
  // код повторяется, пока условие true
}
```

Важно: внутри цикла должно происходить что-то, что когда-нибудь остановит цикл. Иначе получится бесконечный цикл.

Подробнее:

- Циклы `while` и `for`: https://learn.javascript.ru/while-for
- Циклы на MDN: https://developer.mozilla.org/ru/docs/Web/JavaScript/Guide/Loops_and_iteration

- если число пользователя меньше секретного — `"Больше"`;
- если число пользователя больше секретного — `"Меньше"`.

В этой задаче нужно использовать цикл `while`.

### Формат ввода

- Программа спрашивает: `"Угадайте число от 1 до 10"`
- Пользователь вводит число.
- Если число не угадано и попытки остались, программа спрашивает снова.

### Что должна сделать программа

1. Создать переменную `secret` со значением `7`.
2. Создать переменную для количества попыток.
3. Спросить число у пользователя.
4. Если число меньше `secret`, вывести `"Больше"`.
5. Если число больше `secret`, вывести `"Меньше"`.
6. Если число равно `secret`, вывести `"Правильно!"`.
7. Если пользователь потратил 5 попыток, вывести `"Попытки закончились."`.

### Форматы вывода

Если число меньше секретного:

```txt
Больше.
```

Если число больше секретного:

```txt
Меньше.
```

Если пользователь угадал:

```txt
Правильно!
```

Если попытки закончились:

```txt
Попытки закончились.
```

### Примеры

Ввод:

```txt
3
9
7
```

Вывод:

```txt
Больше.
Меньше.
Правильно!
```

Ввод:

```txt
1
2
3
4
5
```

Вывод:

```txt
Больше.
Больше.
Больше.
Больше.
Больше.
Попытки закончились.
```

### Что нужно использовать

- `let secret = 7`
- `Number(prompt(...))`
- `while`
- счётчик попыток
- `if / else if / else`
- сравнение чисел
- `console.log()`
